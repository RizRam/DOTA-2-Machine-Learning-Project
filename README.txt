1.  Install Latest version of Anaconda (https://anaconda.org/)
2.  Use a Python IDE (Spyder3 recommended)
3.  Open a project and navigate to DOTA2ML folder.
    a.) if project does not exist, create a new project and set the directory to the DOTA2ML
4.  Run Preprocessing.py FIRST!
5.  Run Aggregate.py SECOND!
6.  Run Normalize.py THIRD!
7.  Run other py files in any order to get the results
    a.) HelperFunctions.py contains helper functions and does not need to be run

8.  Results.xslx contains all the results as well as charts
